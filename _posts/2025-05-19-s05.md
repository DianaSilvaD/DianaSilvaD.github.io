#### Asignar ip estatica **`192.168.0.4`** a eth1
- Verificar nombre de interfaz
```bash
ip a
```
- Editar la configuración:
```bash
sudo nano /etc/network/interfaces
```
- Modificar la IP
```bash
    auto lo
    iface lo inet loopback

    auto eth1
    iface eth1 inet static
        address 192.168.0.4
        netmask 255.255.255.0
```

#### Acceder al repositorio clonado

- Ingresar a **`wizard_spider`** del repositorio clonado
        
    ```bash
        root@attacker:~/Descargas/adversary_emulation_library/wizard_spider#         
    ```
            
- Instalar **`pyminizip`** para descomprimir los archivos zip.
        
    ```bash
        pip3 install pyminizip
    ```
    ![image.png](/assets/images/2025-05-19-s05/Captura desde 2025-06-06 23-52-43.png)
        
        
- Descifrar el script
        
    El siguiente script para descifra automáticamente estos archivos:
            
    ```bash
        root@attacker:~/Descargas/adversary_emulation_library/wizard_spider# python3 Resources/Utilities/crypt_executables.py -i ./ -p malware --decrypt
    ```
    ![image.png](/assets/images/2025-05-19-s05/Captura desde 2025-06-07 00-24-43.png)  

    ![image.png](/assets/images/2025-05-19-s05/Captura desde 2025-06-07 00-27-02.png)  



- Conectarse al Domain Controller vía RDP

```bash
xfreerdp +clipboard /u:oz\\administrator /p:"P@ssw0rd123" /v:10.0.0.4 /drive:X,wizard_spider/Resources/setup /cert-ignore
```
